# 5. Documentação da API Binance (Futures - Long/Short Ratio)

Para obter o Long/Short Ratio (LSR) dos traders de Futuros na Binance, é necessário consultar a documentação da API de Derivativos (Futures).

## Dados Principais

*   **Long/Short Ratio (LSR):** Proporção de contas ou posições Long/Short entre os traders.
*   **Dados:** LSR de Contas ou LSR de Posições (dependendo do endpoint).

## Autenticação

O acesso a estes dados de mercado de Futuros é **público** e **não requer autenticação**.

*   **Endpoint Base (USDⓈ-M Futures):** `https://fapi.binance.com`

## Endpoints Chave (LSR)

### 1. Long/Short Ratio de Contas de Top Traders

Retorna a proporção de contas net Long e net Short dos top 20% de usuários com o maior saldo de margem.

*   **Método:** `GET`
*   **Endpoint:** `/futures/data/topLongShortAccountRatio`
*   **Parâmetros:** `symbol`, `period`, `limit`.

### 2. Long/Short Ratio de Posições de Top Traders

Retorna a proporção de posições Long e Short dos top 20% de usuários com o maior saldo de margem.

*   **Método:** `GET`
*   **Endpoint:** `/futures/data/topLongShortPositionRatio`
*   **Parâmetros:** `symbol`, `period`, `limit`.

### 3. Long/Short Ratio de Posições de Todos os Usuários

Retorna a proporção de posições Long e Short de todos os usuários.

*   **Método:** `GET`
*   **Endpoint:** `/futures/data/globalLongShortAccountRatio`
*   **Parâmetros:** `symbol`, `period`, `limit`.

## Notas Importantes

*   Estes endpoints são para **USDⓈ-M Futures** (Futuros margem em USDT/BUSD).
*   Apenas dados dos últimos **30 dias** estão disponíveis.
*   O `period` (período) pode ser, por exemplo, `5m`, `15m`, `30m`, `1h`, `2h`, `4h`, `6h`, `12h`, `1d`.
