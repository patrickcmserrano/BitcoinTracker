# 4. Documentação da API Binance (Spot Market Data)

A API Spot da Binance é uma interface pública e robusta para acessar dados de mercado em tempo real e históricos, como preços e dados de velas (OHLCV).

## Dados Principais

*   **Preços BTC/USDT:** Preços em tempo real e estatísticas de 24 horas.
*   **OHLCV (K-lines):** Dados de velas (Open, High, Low, Close, Volume) em vários intervalos de tempo.

## Autenticação

Acesso a dados de mercado (Market Data) é **público** e **não requer API Key**.

*   **Endpoint Base:** `https://api.binance.com` (para a maioria dos endpoints Spot)

## Endpoints Chave (Market Data)

### 1. Klines/Candlestick Data (OHLCV)

Endpoint para obter dados de velas (OHLCV) para um símbolo específico.

*   **Método:** `GET`
*   **Endpoint:** `/api/v3/klines`
*   **Parâmetros Mandatórios:**
    *   `symbol`: O par de negociação (e.g., `BTCUSDT`).
    *   `interval`: O intervalo da vela (e.g., `1m`, `1h`, `1d`).

| Intervalo | Valor do `interval` |
| :--- | :--- |
| Minutos | `1m`, `3m`, `5m`, `15m`, `30m` |
| Horas | `1h`, `2h`, `4h`, `6h`, `8h`, `12h` |
| Dias | `1d`, `3d` |
| Semanas | `1w` |
| Meses | `1M` |

### 2. Preço do Símbolo

Endpoint para obter o preço mais recente de um ou todos os símbolos.

*   **Método:** `GET`
*   **Endpoint:** `/api/v3/ticker/price`
*   **Parâmetro Opcional:** `symbol` (Se omitido, retorna todos os preços).

### 3. Estatísticas de 24 horas

Endpoint para obter estatísticas de mudança de preço em 24 horas.

*   **Método:** `GET`
*   **Endpoint:** `/api/v3/ticker/24hr`
*   **Parâmetro Mandatório:** `symbol` (e.g., `BTCUSDT`).
