# 1. Documentação da API Coinglass (Dados de Derivativos)

A API Coinglass é um sistema robusto para agregar dados de mercado de criptomoedas em tempo real e histórico, com foco em derivativos. É uma ferramenta essencial para análise quantitativa de mercados de futuros e opções.

## Dados Principais

*   **LSR (Long/Short Ratio):** Proporção entre posições long e short.
*   **Open Interest (OI):** O número total de contratos de derivativos pendentes que não foram liquidados.
*   **Liquidações:** Dados detalhados sobre liquidações de posições.
*   **Heatmap de Liquidação:** Visualização de áreas de preço com alta concentração de liquidações.

## Autenticação

A API Coinglass **requer autenticação via API Key**.

> "Para começar, faça login na sua conta e gere sua API Key no seu Painel de API Key."

*   **Endpoint Base:** `https://open-api-v4.coinglass.com`
*   **Método de Autenticação:** A chave deve ser incluída no cabeçalho ou na URL da requisição, conforme as instruções da documentação oficial [1].

## Endpoints Chave (Exemplos)

A documentação aponta para endpoints específicos para os dados solicitados:

| Endpoint (Exemplo) | Descrição |
| :--- | :--- |
| `/public/v2/open_interest` | Retorna dados de Open Interest. |
| `/reference/liquidation-history` | Acesso a dados históricos de liquidação. |
| `/reference/oi-ohlc-aggregated-history` | Dados OHLC (Open, High, Low, Close) de Open Interest agregado entre exchanges. |

## Notas Importantes

*   A API agrega dados de grandes exchanges como Binance, OKX, Bybit, Coinbase, etc.
*   A plataforma expandiu a cobertura para incluir dados de spot, opções, ETF e on-chain.
