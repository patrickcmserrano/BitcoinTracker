# 2. Documentação da API Alternative.me (Fear & Greed Index)

A API Alternative.me fornece acesso ao popular **Crypto Fear & Greed Index (FNG)**, que mede o sentimento do mercado de Bitcoin e outras criptomoedas em uma escala de 0 (Extreme Fear) a 100 (Extreme Greed).

## Dados Principais

*   **Fear & Greed Index (FNG):** Valor numérico de 0 a 100.
*   **Classificação:** Categoria de sentimento (e.g., Fear, Neutral, Greed).
*   **Timestamp:** Data e hora da medição.
*   **Time Until Update:** Tempo restante para a próxima atualização (disponível apenas para o valor mais recente).

## Autenticação

Esta é uma **API pública** e **não requer API Key** ou autenticação.

*   **Endpoint Base:** `https://api.alternative.me/`
*   **Endpoint FNG:** `/fng/`

## Parâmetros Opcionais

O endpoint `/fng/` aceita os seguintes parâmetros de consulta:

| Parâmetro | Tipo | Descrição | Valores Possíveis |
| :--- | :--- | :--- | :--- |
| `limit` | `int` | Limita o número de resultados retornados. | Padrão: `1` (último valor). Use `0` para todos os dados disponíveis. |
| `format` | `string` | Formato da seção de dados. | `json` (Padrão) ou `csv`. |
| `date_format` | `string` | Formato da data. | `us`, `cn`, `kr`, `world` (DD/MM/YYYY) ou vazio (Unix timestamp - Padrão). |

## Exemplo de Requisição e Resposta

**Exemplo de URL:** `https://api.alternative.me/fng/?limit=2`

**Resposta (JSON):**

```json
{
	"name": "Fear and Greed Index",
	"data": [
		{
			"value": "40",
			"value_classification": "Fear",
			"timestamp": "1551157200",
			"time_until_update": "68499"
		},
		{
			"value": "47",
			"value_classification": "Neutral",
			"timestamp": "1551070800"
		}
	],
	"metadata": {
		"error": null
	}
}
```
