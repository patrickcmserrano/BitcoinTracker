# 6. Documentação da API Alpha Vantage (Índice Dólar - DXY)

A Alpha Vantage oferece APIs para dados de mercado de ações, Forex, commodities e criptomoedas. É uma alternativa para obter dados do Índice Dólar (DXY).

## Dados Principais

*   **Preço DXY:** O DXY é tratado como um par de câmbio (Forex).
*   **Dados Históricos:** Séries temporais diárias, semanais ou mensais.

## Autenticação

A API Alpha Vantage **requer uma API Key** (gratuita).

*   **Método de Autenticação:** A chave deve ser incluída como um parâmetro de consulta (`apikey`) em todas as requisições.
*   **Endpoint Base:** `https://www.alphavantage.co/query?`

## Endpoints Chave (Forex - FX)

O DXY (US Dollar Index) é um índice que mede o valor do dólar americano em relação a uma cesta de moedas estrangeiras. Embora não haja um endpoint direto para o DXY, a API de Forex (FX) pode ser utilizada para obter dados de pares de moedas que compõem o índice, ou a biblioteca `yfinance` pode ser usada como alternativa (ver seção 7).

### 1. Taxa de Câmbio em Tempo Real

*   **Função:** `CURRENCY_EXCHANGE_RATE`
*   **Descrição:** Retorna a taxa de câmbio em tempo real entre duas moedas.
*   **Exemplo:** `...function=CURRENCY_EXCHANGE_RATE&from_currency=USD&to_currency=EUR&apikey=YOUR_KEY`

### 2. Séries Temporais Diárias de FX

*   **Função:** `FX_DAILY`
*   **Descrição:** Retorna a série temporal diária (OHLC) para um par de moedas.
*   **Exemplo:** `...function=FX_DAILY&from_symbol=USD&to_symbol=EUR&apikey=YOUR_KEY`

## Notas Importantes

*   Para obter dados que simulem o DXY, pode ser necessário construir um índice ponderado usando os pares de moedas disponíveis na API (EUR/USD, JPY/USD, GBP/USD, CAD/USD, SEK/USD, CHF/USD).
*   A alternativa mais simples para o DXY é usar o wrapper **yfinance** com o símbolo apropriado.
