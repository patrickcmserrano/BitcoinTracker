# 8. Documentação da API Trading Economics (Calendário Econômico)

A API Trading Economics oferece acesso a milhões de linhas de dados históricos, indicadores macroeconômicos e o calendário econômico em tempo real.

## Dados Principais

*   **Eventos Econômicos:** Cerca de 1600 eventos econômicos por mês.
*   **Indicadores Macro:** Dados atuais e históricos de mais de 15.000 séries para 196 países.
*   **Dados do Calendário:** Valores `Actual` (Real), `Previous` (Anterior), `Forecast` (Previsão do Consenso) e `TEForecast` (Previsão da Trading Economics).

## Autenticação

A API **requer API Key** e o acesso completo geralmente está associado a um **plano pago**.

*   **Método de Autenticação:** A chave deve ser incluída na URL da requisição.
*   **Exemplo de Autenticação (Requests Python):**
    ```python
    import requests
    api_key = 'YOUR_API_KEY'
    url = f'https://api.tradingeconomics.com/calendar?c={api_key}'
    data = requests.get(url).json()
    ```

## Endpoints Chave (Calendário Econômico)

O endpoint `/calendar` permite filtrar o calendário econômico.

### 1. Eventos por País, Data e Importância

É possível filtrar o calendário por diversos critérios, sendo os mais úteis:

*   **Por País:** `/calendar/events/country/{país}`
*   **Por Data:** `/calendar/country/All/{data_inicial}/{data_final}`
*   **Por Importância:** Parâmetro `importance` (1-Low, 2-Medium, 3-High).

**Exemplo de Requisição (Por Data e Importância):**

`https://api.tradingeconomics.com/calendar/country/All/2016-12-02/2016-12-03?c={api_key}&importance=3`

## Campos de Resposta (Tabela)

A resposta do calendário econômico é detalhada e inclui campos como:

| Campo | Descrição |
| :--- | :--- |
| `Date` | Data e hora do evento. |
| `Country` | País do evento. |
| `Event` | Nome do evento (e.g., Non Farm Payrolls). |
| `Actual` | Valor real divulgado. |
| `Previous` | Valor anterior. |
| `Forecast` | Previsão do consenso. |
| `Importance` | Nível de importância (1, 2 ou 3). |
