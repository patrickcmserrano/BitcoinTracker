# 9. Documentação da Biblioteca technicalindicators (TypeScript/JavaScript)

`technicalindicators` é uma biblioteca JavaScript/TypeScript para análise técnica, projetada para ser usada tanto no servidor (Node.js) quanto no navegador. É uma alternativa moderna e robusta ao `talib`.

## Dados Principais

A biblioteca suporta mais de 30 indicadores técnicos e padrões de candlestick, incluindo:

*   **Momentum:** MACD, RSI, Stochastic Oscillator (KD), Stochastic RSI (StochRSI).
*   **Tendência:** Médias Móveis (SMA, EMA, WMA), Parabolic SAR (PSAR).
*   **Volatilidade:** Bollinger Bands (BB), Average True Range (ATR).
*   **Volume:** Accumulation Distribution Line (ADL), On Balance Volume (OBV).
*   **Padrões de Candlestick:** Abandoned Baby, Engulfing Patterns, Hammer, Shooting Star, etc.

## Uso (TypeScript/JavaScript)

A biblioteca pode ser usada de duas formas principais:

### 1. Uso com funções globais (menor verbosidade)

```javascript
// Exemplo: Cálculo de SMA
const input = {
  period: 5,
  values: [1, 2, 3, 4, 5, 6, 7, 8, 9],
  reversedInput: true // Opcional
};

const smaResult = sma(input);
// smaResult: [3, 4, 5, 6, 7]
```

### 2. Uso com a função `calculate` (mais explícito)

```javascript
import { SMA } from 'technicalindicators';

const input = {
  period: 5,
  values: [1, 2, 3, 4, 5, 6, 7, 8, 9]
};

const smaResult = SMA.calculate(input);
// smaResult: [3, 4, 5, 6, 7]
```

### Exemplo de MACD

O MACD (Moving Average Convergence Divergence) requer três parâmetros de período:

```javascript
import { MACD } from 'technicalindicators';

const macdInput = {
  values: [/* array de preços de fechamento */],
  fastPeriod: 12,
  slowPeriod: 26,
  signalPeriod: 9,
  SimpleMA: false // Usar EMA (padrão)
};

const macdResult = MACD.calculate(macdInput);
/* macdResult:
[
  { MACD: 0.123, Signal: 0.150, Histogram: -0.027 },
  // ...
]
*/
```
