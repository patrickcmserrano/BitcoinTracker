# 7. Documentação da API Yahoo Finance (yfinance - Alternativa para DXY)

O `yfinance` é um popular **wrapper Python não oficial** que permite baixar dados financeiros do Yahoo! Finance de forma simples e eficiente. É a alternativa mais prática para obter dados do Índice Dólar (DXY).

## Dados Principais

*   **Índice DXY:** Preços históricos e atuais.
*   **Índices Tradicionais:** S&P 500, Dow Jones, etc.
*   **Forex:** Pares de moedas.
*   **Ações e Criptomoedas:** Dados OHLCV e informações de empresas.

## Autenticação

O `yfinance` acessa dados públicos do Yahoo! Finance e **não requer API Key**.

## Símbolos Chave

O principal desafio é identificar o símbolo correto para o DXY no Yahoo! Finance.

*   **Símbolo Comum para DXY:** `DX-Y.NYB` ou `^NYICDX`

## Uso (Python)

A biblioteca é instalada via `pip`: `pip install yfinance`.

```python
import yfinance as yf

# 1. Obter dados do índice DXY
dxy_ticker = yf.Ticker("DX-Y.NYB")

# 2. Obter dados históricos (OHLCV)
# period: '1d', '5d', '1mo', '3mo', '6mo', '1y', '2y', '5y', '10y', 'ytd', 'max'
# interval: '1m', '2m', '5m', '15m', '30m', '60m', '90m', '1h', '1d', '5d', '1wk', '1mo', '3mo'
hist_data = dxy_ticker.history(period="max")

# 3. Obter informações em tempo real (último preço)
# O último preço pode ser obtido do DataFrame retornado ou de outras propriedades do Ticker.
latest_info = dxy_ticker.info
latest_price = latest_info.get('regularMarketPrice')

print(f"Último Preço do DXY: {latest_price}")
print(hist_data.tail())
```
