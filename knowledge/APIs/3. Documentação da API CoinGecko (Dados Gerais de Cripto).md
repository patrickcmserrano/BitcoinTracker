# 3. Documentação da API CoinGecko (Dados Gerais de Cripto)

A CoinGecko é um dos maiores agregadores de dados de criptomoedas, fornecendo uma ampla gama de informações de mercado, metadados e dados históricos.

## Dados Principais

*   **BTC Dominance:** Percentual da capitalização de mercado do Bitcoin em relação ao total.
*   **Market Cap:** Capitalização de mercado total e por moeda.
*   **Preços e Rankings:** Preços em tempo real, volume, e rankings.

## Autenticação e Limitações

*   **Plano Gratuito (Demo API):** Disponível, mas com **rate limits** (limites de requisição).
*   **Endpoint Base (Demo):** `https://api.coingecko.com/api/v3/`
*   **Autenticação:** A chave de demonstração (Demo API Key) é necessária e pode ser fornecida de duas maneiras:
    1.  **Header (Recomendado):** `x-cg-demo-api-key: YOUR_API_KEY`
    2.  **Query String Parameter:** `x_cg_demo_api_key=YOUR_API_KEY`

## Endpoints Chave (Exemplos)

### 1. Dados Globais (BTC Dominance e Market Cap Total)

O endpoint `/global` fornece dados globais do mercado de cripto, incluindo a porcentagem de capitalização de mercado por moeda (Dominância) e a capitalização total.

*   **Método:** `GET`
*   **Endpoint:** `/global`
*   **Dados Retornados:** O campo `market_cap_percentage` contém a dominância do BTC.

```json
{
  "data": {
    // ... outros campos
    "total_market_cap": {
      "usd": 2721226850772.63,
      // ... outras moedas
    },
    "market_cap_percentage": {
      "btc": 50.4465263233584, // Dominância do BTC
      "eth": 14.9228066918211,
      // ... outras moedas
    },
    // ... outros campos
  }
}
```

### 2. Preços e Rankings

A seção **Coins** da API Demo oferece endpoints para listar moedas, obter dados de mercado e histórico.

*   **Endpoint:** `/coins/markets` (Para preços e rankings)
*   **Endpoint:** `/coins/{id}/market_chart` (Para dados históricos de preço, market cap e volume)
